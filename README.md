## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://maker.makecode.com/](https://maker.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/lizzymejia/mejia_lizzy_light-sensor** and import

## Edit this project ![Build status badge](https://github.com/lizzymejia/mejia_lizzy_light-sensor/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://maker.makecode.com/](https://maker.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/lizzymejia/mejia_lizzy_light-sensor** and click import
